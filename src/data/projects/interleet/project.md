## Overview

Interleet is a collection of interactive visualizations for LeetCode problems. Reading a solution is not the same as understanding it. You can follow code line-by-line but still have no intuition for why it works. Watching it run is a different experience.

Each problem has an animated walkthrough: data structures updating, pointers moving, decisions branching. You get to see the algorithm behave rather than simulate it in your head.

Check it out at the project link above, or [here](https://interleet.rkvempati.com/).

## Open Contributions

I opened Interleet to contributions. A lot of problems don't have a visualization yet, and there's room for creative takes on ones that do. If you have an idea for how to make a problem click visually, feel free to add it.
